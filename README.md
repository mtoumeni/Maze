# Maze
Maze Game (jeu de labyrinthe)
Welcome to Maze game.
To win, the player must reach the letter "U".
He can cross the dot "." and the empty space, but cannot cross the letter "O" which here represents a wall. 
The commands are as follows:
ni -> we move North
si -> we move South
wi -> we move West
ei -> we move East
i represents the number of moves.
Examples:
n1 we move one square north
s2 we move two squares south
w5 we move five squares west
e4 we move four squares east\n
The maze is only saved if you haven't won. You can continue next time.
You must choose a maze at the first start or after a victory.

------ french version -------
Bienvenue dans le jeu du Labyrinthe.
Pour gagner, le joueur doit atteindre la lettre « U ».
Il peut franchir le point « . » et les espaces vides, mais pas la lettre « O », qui représente ici un mur.
Les commandes sont les suivantes :
ni : déplacement vers le Nord
si : déplacement vers le Sud
wi : déplacement vers l'Ouest
ei : déplacement vers l'Est
i représente le nombre de déplacements.
Exemples :
n1 : déplacement d'une case vers le Nord
s2 : déplacement de deux cases vers le Sud
w5 : déplacement de cinq cases vers l'Ouest
e4 : déplacement de quatre cases vers l'Est
La progression dans le labyrinthe est sauvegardée uniquement si vous n'avez pas encore gagné. Vous pourrez continuer la partie suivante.
Vous devez choisir un labyrinthe au début de votre première partie ou après une victoire.
